NERU

Description

Neru is a Discord bot designed to provide a variety of useful functions and features within the Discord servers. With its wide range of commands and capabilities, Neru stands out as a virtual assistant to ease administration and improve the user experience on the server.

Some of Neru's main features include:

Moderation

Neru provides tools to moderate and maintain a safe server environment. You can perform actions such as ban, kick or silence problematic users. It also offers functions to manage word filters, control the flow of messages and detect spam.

Server Information

Neru may provide detailed information about the server, such as user statistics, activity levels, and moderation logs. You can also display information about roles, channels, and custom emojis.
